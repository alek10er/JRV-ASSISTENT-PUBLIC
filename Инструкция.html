Очень легко настраиваемая программа с гибким интерфейсом, не требующая знаний программирования

1. Настройка фраз активации и деактивации

Вы можете настроить слова активации, на которые Джарвис будет реагировать, а также фразы деактивации.

Как настроить:

Откройте папку configs и в ней файл phrases.json. Замените любую фразу активации на свою или добавьте новую.

Важно: У последней фразы в списке не должно быть запятой, а у предыдущих должна быть.

Пример:

{
  "activation_phrases": [
    "джарвис",
    "привет джарвис",
    "эй джарвис"
  ]
}

То же самое можно сделать с фразами деактивации.

2. Исправление ошибок распознавания речи

Если при распознавании речи возникают ошибки, вы можете добавить исправления.

Как настроить:

Зайдите в папку configs, откройте файл corrections.json и добавьте фразу и соответствующую команду.

3. Настройка звуков

Если вам надоела озвучка Джарвиса, вы можете заменить звуковые файлы.

Как настроить:

Зайдите в папку configs и откройте файл sounds.json. Замените названия звуковых файлов на свои.

Пример:

{
  "activation": "66756.wav", // активация
  "success": [ // успешное выполнение действия
    "success1.wav",
    "success2.wav"
  ],
  "error": "error.wav", // неизвестная команда или неудачное распознавание речи
  "deactivation": "deactivation.wav", // звук на отключение
  "voice_activated": "voice_on.wav" // звук на активацию после деактивации
}

4. Интерфейс программы

В программе уже есть легкие настройки интерфейса:

Панель управления

Показывает последние 10 логов. Также можно проверить команды на работоспособность.

Голосовое управление

Можно посмотреть ряд обычных команд и принудительно активировать голосовое управление.

5. Команды

Можно создать несколько видов команд без знания программирования:

Запуск программы

Выберите программу и напишите текст активации, который вы будете говорить для запуска программы.

Веб-сайты

Укажите текст команды и ссылку на сайт, который нужно открыть.

Горячие клавиши

Настройте горячие клавиши, которые будут выполняться при произнесении команды.

6. Настройки

В настройках можно настроить следующие параметры:

  • Распознавание речи
  • Время ожидания речи (сколько секунд)
  • Максимальная длительность фразы до тайм-аута
  • Потребление мощностей

7. Логи

В разделе "Логи" можно посмотреть все действия программы.

Важно: Программа не работает без интернета.
